MacOS 操作笔记

发表于 2019-03-13 更新于 2026-05-25 分类于 指南

No MacBook，记录测试 MacOS 效果的笔记。主要包括：安装、主要工具。

安装

  • 安装包：macOS 10.12 Sierra Final by TechReviews.vmdk
  • 环境：Virtualbox

  1. 下载及解压 macOS 10.12 Sierra Final by TechReviews.vmdk

  2. 下载及安装 VirtualBox-5.2.0

  3. 在 VirtualBOX 新建虚拟电脑，内存分配最少 2048 MB 以上
    名称 : OSX10.12
    类型 : Mac OS X
    版本 : Mac OS X 10.12(64 bit)

  4. 点选 使用已有的虚拟硬盘文件, 选择种子下载及解压后的文件 OS X 10.11 El Capitan GM by TechReviews.vmdk

  5. 打开 VirtualBOX 虚拟机的设置
    检查及修改系统主板, 去掉 “ 软驱 “，硬盘选择上移，芯片组改为 ICH9 ，
    系统 -> 处理器，如果你的机器是 4 核,可选择双核 CPU 数量 = 2

  6. 显示 -> 显存大小设置到最大 128 MB

  7. 鼠标右键单击 并选择以系统管理员身份再执行 VirtualBOX
    3 在 Windows 搜索 cmd , 鼠标右键单击选择以系统管理员身份执行 cmd 命令行
    4 请根据 ⑶.1 虚拟电脑的名称 OSX10.12，在 cmd 命令行依次输入命令如下：

由于虚拟机不支持 Apple Quartz Extreme/Core Image，需要 Quartz Extreme 的应用软件例如 iBooks Author，Pixelmator，SketchBook 等不能在虚拟机下使用 。

vboxmanage setextradata “OSXElCapitan” “VBoxInternal/Devices/efi/0/Config/DmiSystemSerial” “C02MD2XNF5V7”
1 关闭 VirtualBOX 及 关闭 cmd 命令行窗口
2 重新正常启动 VirtualBOX, 及启动 OSXElCapitan 虚拟机文件
3 进入 Mac OS X 后，开始设置国家
4 如果没有 Apple ID 可选择不登录
5 输入用户名称和用户初始密码
6 设置时区
7 完成其他安装步骤后，并成功进入 Mac OS X 系统，开始系统偏好设置, 更改语言 (左上角的  苹果菜单 -> 系统偏好设置 (System Preferences) -> Language & Text, 重新启动后才会更新
