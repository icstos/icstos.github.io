概述

Manjaro Linux：一款基于 Arch Linux、对用户友好、全球排名第 1 的 Linux 发行版（排名数据源于 DistroWatch ）。最适合我的的 Liunx 系统。

特点

  • 滚动更新，最新体验，我要最新技术；
  • 具有测试过的稳定软件包的软件仓库里 + AUR 软件仓库：世界上最齐全的 Linux 软件仓库，我要最全的软件；
  • wiki+ 活跃的社区，遇到的问题基本能被解决；
  • 相比于 ArchLinux 更用户友好，对显卡的硬件检测、支持更好，比 ArchLinux 更实用;
  • 快速、强大、轻量、用户友好、高可用性，强！强！强！

桌面系统选择

  • kde 开机：占用内存：650mb 左右
  • xfce 开机：20200517 测试：初始 680mb，卸载一些不必要的（如蓝牙）后 523 mb，BUT
    • 字体：设计的人没有事先安装看一下吗？这默认字体能看？
    • 设置界面、左下角用户启动器、各主题…
    • 丑 拒，别说现在 523 mb，安装完各种软件和环境后怕也得 600 +，就算 KDE 比你多个一两百 mb，我也要选 KDE 了。
    • AND
  • KDE、KDE、KDE
    • 够 Linux，定制化强，方便修改
    • 首选
    • 自由度是真的高，设置起来可以不用鼠标了
    • 用了几年的 xfce，kde 配置起来真的爽
  • 曾经使用过的xfce
    • 相比于 KDE 与 GNOME，轻量级，省资源
    • 易使用，一定的配置性，快捷键不错
    • 稳定，优雅
    • 终端简洁好用
    • 一度纠结用 i3wm，考虑兼容性、xfce 的 manjaro 官方原配性，还是选择了 Xfce
  • 网友的说法
    • “xfce 轻量，不吃性能。kde 华丽，性能尚可。gnome 简约，占内存最多、最卡。”
    • “ 废话：8G 以上内存的，爱谁用谁。本人 cinnamon”

官方推荐的

  • XFCE：非常轻量，适合硬件条件比较差的用户，占用资源较少（现在 KDE 优化过了，好像不比它差）
    • 体现了模块化和可重用的传统 UNIX 哲学。
  • KDE：操作界面类似 Windows 的操作界面
    • Plasma Desktop 提供现代桌面环境体验所需的所有工具，所以你可以从开始就充满生产力。
  • ARCHITECH：命令行版本，类似 Arch Linux，后期要装什么都自己解决，适合 Geek
  • GNOME（GNOME3）：操作界面类似 Mac OS。
    • GNOME Flashback - GNOME Flashback 是 GNOME 3 的一个 shell，最初叫 GNOME fallback 模式。桌面的布局和底层技术都与 GNOME 2 相似。
    • Gnome - GNOME 桌面环境是一个有吸引力且直观的的桌面，同时有现代（GNOME）和经典（GNOME Classic）会话模式。

社区维护的版本

  • MATE：GNOME2 的延续，用不惯 GNOME3 的可以选择使用
    • Mate - Mate 用传统隐喻给 Linux 用户提供一个直观又有吸引力的桌面。MATE 是 GNOME 2 的一个 fork。
  • CINNAMON：Mint Linux 使用的界面，使用了 GNOME3 的技术，实现了 GNOME2 的界面
    • Cinammon - Cinnamon 努力提供一种传统的用户体验。Cinnamon 是一个 GNOME 3 的 fork。
  • OPENBOX：同样一个轻量级的桌面，没有使用过，具体不太了解。
  • AWESOME：高度可定制化的一个桌面系统，适合 Geek
  • BSPWM：轻量平铺窗口的桌面，没用过。
  • BUDGIE：看界面像是基于 GNOME2
    • Budgie - Budgie 是一个面向现代用户思想设计的桌面环境，专注于简洁与优雅。
  • I3：高度可配置的平铺化界面
  • DEEPIN：国内团队开发，界面介于 Windows 和 Mac OS 之间。UI 看起来不错

其他的

  • LXDE - Lightweight X11 Desktop Environment （轻量化 X11 桌面环境）是一个快速且节能的桌面环境。
  • LXQt - LXQt 是 LXDE 的 Qt 移植和即将到来的版本，轻量化的桌面环境。
  • Pantheon - Pantheon 是最初为发行版 elementary OS 创建的桌面环境，同时也是它的默认桌面环境。
  • Unity - Unity 是 GNOME 的一个 shell，由 Canonical 为 Ubuntu 设计。 guide on how to install unity 8 on Ubuntu 16.04.
